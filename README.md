## hey, i'm luffyyy

i make stuff for PAYDAY 2, mostly modding tools and UI improvements. been doing this for a while now.

### what i work on

**BeardLib Editor** - the main thing i'm known for. it's an editor for PAYDAY 2 that lets you create custom heists and maps. written in Lua and uses the BeardLib framework. pretty much the standard tool for PD2 modders at this point.

**HoloUI** - a cleaner UI overhaul for PAYDAY 2. no clutter, just better menus.

**DieselBundleViewer** - a tool for viewing and extracting stuff from PAYDAY 2's bundle files. if you mod the game, you've probably used this.

i also mess around with:
- C# tools for the Diesel engine (PD2's engine)
- C++ when i need performance
- JavaScript/Vue for web stuff
- some Python and Java here and there

most of my repos are PAYDAY 2 related - tools, HUD mods, editors, documentation. if you're into PD2 modding, feel free to check them out or reach out.
